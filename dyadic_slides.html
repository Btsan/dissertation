Dyadic Intervals

A Hierarchical Partitioning Scheme

What are Dyadic Intervals?

A dyadic interval at level J is:

[k · 2J, (k+1) · 2J]
  • J = level (determines interval size: 2J)
  • k = index (which interval at that level)
  • Higher J = larger intervals, fewer partitions

Key Properties

  • Hierarchical - Each level doubles the interval size
  • Complete Coverage - Every point is in exactly one interval per level
  • Non-overlapping - Intervals at same level don't overlap
  • Binary Tree Structure - Each interval contains two child intervals from level below

J=3: interval [0, 8), k=0

Path through hierarchy:
5 → 2 → 1 → 0

Applications

  • Segment Trees - Efficient range queries
  • Range Sketches - Approximate query processing
  • Wavelet Analysis - Signal processing
  • Hierarchical Decomposition - Multi-resolution data structures
